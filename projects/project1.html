Анализ плотности населения

Интерактивная карта показывает распределение населения в выбранном районе. Нажмите на любой хексагон, чтобы увидеть точное количество жителей.

О проекте

Для этого проекта использовались данные Kontur Population (H3-хексагоны с населением). Каждый хексагон окрашен в зависимости от плотности населения: светлый — низкая плотность, темный — высокая.

Технологии: Maplibre GL JS, GeoJSON, HTML/CSS, JavaScript
